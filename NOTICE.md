## Propósito de este repositorio

`adlc-kit` se distribuye como **kit de inicio** para equipos y
contribuidores que quieran arrancar su propio **ciclo de desarrollo
agéntico** sobre la base del flujo ADLC original. Incluye:

- Los **agent_specs** canónicos del flujo de 9 pasos
  (Discovery → Hypothesis → Mapping → Spec Development → Architecture
  → Business Case → Orchestrator → Coding Agents → Validation).
- El **engine** Python (FastAPI + Postgres + sandbox Docker).
- La **UI** React para operar runs, aprobar HITL y editar specs.
- Un `infra_constraints/machbank.yaml` como ejemplo de stack target
  (MACHBank edition).
